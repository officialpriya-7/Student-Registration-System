# Student Registration System

A responsive web-based Student Registration System built using **HTML**, **CSS**, and **JavaScript (DOM Manipulation)**. This project allows users to register students, edit and delete records, and stores all entries using **localStorage** so data persists after page refresh.

---
##  Features
- Add new student records
- Edit existing records
- Delete records with confirmation
- Validation for:
  - Only characters allowed in student name
  - Only numbers allowed in student ID and contact number
  - Contact number must have at least 10 digits
  - Valid email format
- Prevents empty or duplicate entries
- Fully responsive (Mobile, Tablet, Desktop)
- Vertical scrollbar added dynamically when the table grows
- Data saved in **localStorage** so it remains after refresh

##  How to Run the Project
1. Download the project or clone it via GitHub.
2. Make sure all files are in the **same folder**.
3. Open **index.html** in any modern browser.

No installation required.

---
##  Testing the System
1. Fill out the registration form.
2. Click **Add Student**.
3. See the record appear in the table.
4. Try editing and deleting records.
5. Refresh the page — records remain (localStorage).
